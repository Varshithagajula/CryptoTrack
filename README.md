## ğŸš€ Getting Started

### Prerequisites

* Node.js (preferably latest LTS)
* npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Varshithagajula/CryptoTrack.git
   cd CryptoTrack
   ```
2. Install dependencies:

   ```bash
   npm install
   ```

   or

   ```bash
   yarn
   ```
3. Start the development server:

   ```bash
   npm run dev
   ```

   or

   ```bash
   yarn dev
   ```
4. Open in your browser: `http://localhost:5173` (or whichever URL Vite outputs).
